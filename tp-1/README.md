# README

## Configuration

Créer un fichier `.env` à la racine du projet avec la clé suivante :

```
PING_LISTENING_PORT=...
INSTANCE_ID= ...
```

## Installation

```bash
npm install
```

## Lancement

```bash
npm run start
```

## Utilisation

Accéder à la route pour voir votre header :

```
/ping
```

et cette route pour accéder a des stats :

```
/stats
```
